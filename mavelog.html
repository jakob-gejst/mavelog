Indsæt din Apps Script URL herover
1

Åbn dit Google Sheets-regneark

Gå til dit eksisterende regneark eller opret et nyt. Klik på Udvidelser → Apps Script.

2

Indsæt dette script

Slet alt i editoren og indsæt koden nedenfor. Klik Gem.

function doPost(e) { const data = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('Mavelog') || SpreadsheetApp.getActiveSpreadsheet() .insertSheet('Mavelog'); if (sheet.getLastRow() === 0) { sheet.appendRow(['Dato','Tidspunkt','Kl.','Score', 'Symptomer','Mad','Kaffe','Søvn score', 'Søvn timer','Stress','Motion','Alkohol','Noter']); } sheet.appendRow([ data.dato, data.tidspunkt, data.klokkeslaet, data.mavescore, (data.symptomer||[]).join(', '), data.mad||'', data.kaffe||'', data.sovn_score||'', data.sovn_timer||'', data.stress||'', data.motion||'', data.alkohol||'', data.noter||'' ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
3

Deploy som webapp

Klik Deploy → Ny deployment. Vælg type: Webapp. Sæt "Kør som" til Mig og "Adgang" til Alle. Klik Deploy og kopiér URL'en.

4

Indsæt URL'en herover

Kopiér din webapp-URL og indsæt den i feltet øverst. Appen sender herefter automatisk alle check-ins til dit regneark.
